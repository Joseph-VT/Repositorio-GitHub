HTML significa "Linguagem de Marcação de HiperTexto" (HyperText Markup Language). Esta seção objetiva abordar os fundamentos de hipertexto e sua relação com a linguagem HTML. A próxima seção aborda as características de HTML como linguagem de marcação.
HTML é uma linguagem de marcação

Tecnicamente, o conceito de hipertexto é bastante simples: é um documento eletrônico composto por páginas e ligações entre elas (links). HTML é uma linguagem para escrever este tipo de documento. Existem outras linguagem para a escrita de hipertexto, mas HTML é uma linguagem específica, adotada como padrão para a escrita de hipertextos a serem publicados na Web (consultar seção anterior, HTML: uma linguagem para Web). Os hipertextos escritos nesta linguagem são denominados sites ou homepages.

Site e homepage são denominação empregadas especificamente para os hipertextos da Web. Em geral, estes termos são utilizados como sinônimos. Mas há diferença sutil entre eles. Homepage refere-se principalmente a um hipertexto sobre alguém, uma empresa ou instituição, e geralmente, carrega em si a idéia de ser um hipertexto pequeno, com poucas páginas. Site é um termo mais genérico sendo utilizado para designar todo tipo de hipertexto (inclusive os pessoais), e geralmente refere-se a hipertextos grandes, com muitas páginas.

Convenções. As etiquetas, os atributos e valores podem ser escritos tanto em maiúsculo como em minúsculo, e até misturado. Desta forma, ou representam a mesma etiqueta.

Observe que no meio deste código encontram-se certos "símbolos estranhos" como P, I, /I, B, /B . Estes símbolos são denominados etiquetas e são utilizados para indicar como o texto será visualizado. Por exemplo, a etiqueta I marca o início do texto em Itálico e a etiqueta /I marca o fim do texto em itálico.

O termo original em inglês, browser, se traduzido ao pé-da-letra, significa "folheador". Esta denominação tem origem na metáfora que estabelece a comparação entre o hipertexto e o livro - como se o usuário, ao transitar entre as páginas do hipertexto, estivesse folheando as páginas de um livro. O termo Navegador tem origem na metáfora que estabelece a comparação entre o ato de transitar entre as páginas do hipertexto e ato de navegar.

Observe que a formatação do texto é obtida com o uso de etiquetas - símbolos da linguagem HTML identificados pelos sinais "< >" (se for a etiqueta inicial) ou pelos sinais "" (se for a etiqueta final). Por exemplo, a etiqueta I marca o início do texto a ser apresentado em itálico, e a etiqueta /I marca o fim da formatação em itálico. Portanto, tudo o que estiver entre este par de etiquetas I e /I será apresentado em itálico. Assim também funcionam as etiquetas B e /B, que marcam o início e o fim do texto a ser apresentado em negrito (Bold).


Etiqueta é a tradução aqui adotada para o termo tag (que é o termo original utilizado nas especificações da linguagem HTML). Alguns autores adotam outras traduções como marcador, rótulo, comando ou mesmo o termo original tag.

A linguagem HTML especifica elementos para construir estruturas como: formatação de texto, parágrafo, lista, tabela, etc. Cada elemento, em geral, é composto por três partes: etiqueta inicial + conteúdo + etiqueta final. O elemento itálico exemplifica esta estrutura genérica:

I ............ Texto .......... /I
Etiqueta _ Conteúdo _ Etiqueta
Inicial..........................Final

A linha horizontal é construída com o elemento vazio HR.

A estrutura principal deve iniciar com a etiqueta HTML e terminar com a etiqueta /HTML. O cabeçalho, delimitado pelas etiquetas HEAD e /HEAD, destina-se a conter informações sobre o documento - por exemplo, o título do documento (que é construído com o elemento TITLE e aparece no título da janela do navegador). O corpo do documento, delimitado pelas etiquetas BODY e /BODY, destina-se ao conteúdo da página. Cabeçalho. Um texto geralmente é organizado em capítulos, seções, tópicos etc. O elemento H (Heading = cabeçalho) pode ser utilizado para construir os cabeçalhos das seções de um texto. Existem seis níveis de cabeçalhos, de H1 (o primeiro e mais importante nível) a H6 (o último e menos importante). O cabeçalho, assim como o parágrafo, também admite o atributo align.

Devo utilizar estilo físico ou lógico? Em geral, o estilo físico é mais utilizado do que o estilo lógico. Com o estilo físico há maior controle sobre a formatação do texto - o que não ocorre com o estilo lógico (que indica o tipo de informação, não a sua aparência).

Código das cores em HTML
Para compreender o código das cores na linguagem HTML, é preciso compreender o sistema RGB.
Sistema RGB

A cor-luz é a cor observada através dos raios luminosos, é a própria luz que pode se decompor em muitas cores. Em relação à cor-luz, as cores primárias são vermelho-alaranjado, verde e azul. Em inglês, estas cores recebem as denominações Red, Green, Blue e constituem o sistema RGB. O monitor do computador trabalha com luz e utiliza este sistema de cores.
